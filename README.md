# SmartMoney-Nexus
"Smart Money" refers directly to the institutional funds, hedge funds, and FIIs that move markets. "Nexus" means a central connection point. This name signals that, this app serves as the ultimate hub where retail traders can track the exact footprints of institutional players.

# Institutional Smart Money & Sector Rotation Tracker

An enterprise-grade, real-time macroeconomic dashboard built with Python and Streamlit. This app scrapes global indexes, tracking macro flows, yield curve health, and automated equity valuations to mirror institutional-grade allocations.

## 🌟 Key Features
- **Synthetic Equity Valuation Index (EVI):** Modeled baseline tracking fair value risk boundaries.
- **Sectoral Relative Rotation Machine:** Maps performance indices into Leading, Weakening, Lagging, and Accumulating (In-Queue) buckets.
- **Yield Curve Monitor:** Measures the 10-Year minus 2-Year bond yields to spot recessions or sudden expansions.
- **Global Ingestion Flow Matrix:** Compares $DXY$ against $EEM$ to see when FIIs are rotating capital back into emerging economies.
